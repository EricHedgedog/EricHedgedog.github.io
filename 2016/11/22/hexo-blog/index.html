hexo + Github 搭建静态博客

2016-11-22

hexo是一个静态博客框架，能够很方便的在github上搭起来个人博客。

  1. 首先要安装node.js

  2. npm install hexo -g 全局安装hexo

  3. 在当前目录下初始化一个博客文件夹myblog ：

    hexo init myblog

    进入博客目录：

    cd myblog

    安装package.jason中的所有依赖包：

    npm install

  4. 在本地启动服务

    hexo server

    启动成功就可以在 localhost：4000 访问了

  5. github 上创建 username.github.io 的仓库，username一定要和自己的 Github 账户名一样。

  6. 在本地博客文件夹找到 _config.yml 文件，找到文件最后的 deploy，在此设置部署的仓库地址。

    type: git
    repository:https://github/EricHedgedog/EricHedgedog.github.io.git
    branch: master

  7. 生产博客静态文件

    hexo g

  8. 把静态文件部署到设置的 Github 仓库（要在 git bash 里执行此命令）

    hexo deploy

此时就可以在 username.github.io 这个地址访问生产的博客页面了。
