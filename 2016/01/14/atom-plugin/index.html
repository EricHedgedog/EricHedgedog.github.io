前段常用 Atom 插件

一些 Atom 常用插件：

  1. Emmet
    你懂的

  2. autoprefixer
    用来补充 CSS 前缀

  3. color-picker
    取色器、

  4. linter
    识别多门语言错误

    • linter-jshint 更细化的针对JS和JSON
    • linter-scss-lint 针对 SASS/SCSS
    • linter-less 针对 LESS
    • linter-csslint 针对 CSS
  5. autocomplete-plus 完善自带的autocomplete
    • autocomplete-paths 路径补全
    • autocomplete-html
    • autocomplete-css

  6. less-autocompile
    自编译

  7. docblockr
    注释插件

  8. file-icons
    增加文件图标

  9. minimap
    右侧迷你代码图导航
